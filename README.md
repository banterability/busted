# Busted

Next stop information for Chicago busses via SMS & Web

## Configuration

Expects an [API key for CTA Bus Tracker](http://www.transitchicago.com/developers/bustracker.aspx) at $BUSTRACKER_API_KEY.

## Usage

### SMS

Text '1366':

```
Rt 8:
In 1m: Milwaukee/Grand (Blue Line)
In 1m: Erie
In 2m: Huron
In 3m: Chicago
In 4m: North Branch
```

### Web

```
/route/1366
```
